# MachineLearning

### 该项目为实现以下统计机器学习算法

实现语言： python3.7

数据在dataset文件夹

算法：
* Kmeans算法
* KNN算法
* Linear Regression算法
* Logistic Regression算法
* Softmax Regression算法
* BP神经网络算法
* CART算法(Classification And Regression Tree)
* 贝叶斯算法（朴素贝叶斯算法）
* PCA算法


数据源有四个：

- abalone.data 适合回归算法
- diabetes.arff 适合二分类算法
- iris.arff  适合多分类算法
- weather.nominal.arff 适合二分类算法
